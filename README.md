# Automation-Testing-Project-Factorial-Website
This project automates regression testing for the Factorial Calculator Web Application using Selenium WebDriver, TestNG, Page Object Model, Data-Driven Testing, and WebDriverWait. The focus of the project is to validate UI elements, functional workflows, and detect bugs in the application.  

**Project Overview**

The Factorial Calculator website allows users to input a number and get the factorial output.
This automation suite verifies:
Core UI components
Correct factorial computation
Error validations
Browser compatibility
End-to-end workflows
Regression stability
